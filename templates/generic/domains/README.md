# Domain Notes

Create one file per major business or technical domain.

Examples:

- `auth.md`
- `billing.md`
- `orders.md`
- `notifications.md`
- `reporting.md`
- `data-pipeline.md`
- `mobile-app.md`
- `admin-ui.md`
- `integrations.md`
- `infrastructure.md`

Keep `.claude/project-map.md` as the short high-level index. Put detailed domain notes here.
